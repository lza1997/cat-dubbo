先说说之前用的NOTE2使用中发现的问题

优点1.刚拿到手的时候专门看了卡槽，比起note1时候已经好很多，拿出来看了下卡槽，很结实，不像上代那么松动。但是手在卡槽闭合的状态下摸过去还是会有凹凸感。这是进步的地方，值得表扬

2.随便玩了下，发现续航确实不错，我一拿到手就开始恢复flyme账户，51%电量，一直在下载同步中，期间下了开心消消乐，玩了大概十分钟，总共使用到现在快一个小时。只是感觉手部微热。现在还有40%的电量，期间各种亮屏，锁屏。本人是mx2老用户，对于这样的续航很满意了。

3.手机拿到手以后，就各种适应，因为本身从MX2过来的，flyme的同步真的很好用。

4.摄像头的问题，我撕掉贴膜以后拍照发现效果还不错，无论对焦的速度，还是拍照的相应速度，机会都没有延迟。

缺点1.玩机过程中发现一个问题，就是在系统设置界面，我频繁的按home键发现听到了home键回弹的声音，但手机界面还保持在系统设置的界面，最明显的就是玩开心消消乐的时候，home按下根本不能回到主页。这个很蛋疼。我只好在home旁上滑把开心消消乐关掉。不知道是哪里的问题
2.玩开心消消乐的时候，能发现很明显的掉帧，卡卡的，不过没有测试其他游戏，不知道别的会不会这样。大家可以试试看是不是个别问题。

3.外放的声音，在我拨号的时候，声音异常小，但是听歌觉得声音还蛮大的，这个有点费解，是我没设置好？

总结，799的魅蓝note2是我现阶段用过的最好用的千元机，但是好产品遇到了成本控制，只能选择妥协，这是在所难免的。最后希望魅族越来越好。
